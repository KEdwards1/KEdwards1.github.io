Projects

Tourism

Built on the Sinatra framework with an Angular.js front-end, Tourism leverages the Songkick and Spotify APIs to allow users to view a visual history of their favourite artist's tour history. Users can also save the setlist for each performance to their Spotify account.

Yafa

Built on the Ruby on Rails framework, Yafa is a web app to help language learners collect materials in their L2 and drill vocabulary. Yafa allows users to upload video, pdf, and audio files and uses Dropbox for file storage. In addition to uploading documents, users can also create lists of vocabulary words and review these lists in a matching game. The mactching game is implemented using HTML, CSS and the JQuery and JQuery UI JavaScript libraries.
